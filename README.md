## Run Locally

Clone the project

```bash
  git clone https://github.com/chasesng/birdsongApp.git
```

Go to the project directory

```bash
  cd .\birdsongApp\
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Features

- Light/dark mode toggle
- Text/Image upload
- Authentication
- Leave likes/comments on posts
